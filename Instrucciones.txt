Para el backend:
npm init
npm i express nodemon mongoose cors morgan body-parser bcryptjs jsonwebtoken mongoose-autopopulate

Para correr el back: npm run dev
_________________
Para el front:
npm i axios sweetalert2 react-router-dom react-chartjs-2

Para correr el front: npm start
